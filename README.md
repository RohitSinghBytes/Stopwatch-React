# ⏱ React Stopwatch

A simple and clean stopwatch application built using **React**. This project was made to practice and understand core React hooks like `useState`, `useRef`, and `setInterval`.

## 🛠️ Built With

- React
- JavaScript (ES6+)
- CSS3

## 🧠 Concepts Used

- `useState` — to track the time and running state
- `useRef` — to store the interval ID without causing re-renders
- `setInterval` / `clearInterval` — to start and stop the timer
- Functional components
